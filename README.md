# weather tracker

## Description
It is an app that help one know about the weather condition of a certain place.


## Features
Search weather by city name
Display temperature,Humidty and windspeed


## Installation
Clone the repository and install dependencies
npm install


## Run
npm run dev

## Technologies
React
Javascript
Tailwindcss

## API used
Open WeatherMap API
